# API Endpoints

## HTML API

### Users

- `GET /user/new`
- `POST /user`
- `PATCH /user`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Stores
- `GET api/stores`
- `GET api/stores/:id`

### Reviews
- `GET api/reviews/:storeid`
- `POST api/reviews/:storeid`

### Pictures
- `GET api/pictures/:storeid`
- `POST api/pictures/:storeid`

### Users
- `GET api/user/:id`
